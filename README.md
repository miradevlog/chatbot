## 🚀 Features

- ⚛️ React (CDN, no build tools)
- 🤖 Simple chatbot logic via `Chatbot.getResponse()`
- 💜 Dark violet glassmorphism UI
- ✨ Smooth hover animations
- 🧊 Frosted glass effects (blur + transparency)
- 💬 Chat bubbles (user vs bot styling)
- 📜 Scrollable chat history
- 🖼️ Avatar support (robot/user images)
- ⚡ Instant response system

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Tailwind CSS v4 via CDN)
- React (CDN version)
- Babel (JSX in browser)
- SupersimpleDev Chatbot API
